# Roadmap

## Overview

Synapse Optical is currently an active proof-of-concept project focused on AI-assisted network operations, deterministic workflow validation, and operational troubleshooting.

This roadmap reflects architectural direction and development focus areas rather than committed release timelines.

---

# Current Proof-of-Concept Focus

## Operational Troubleshooting

- VPN troubleshooting workflows
- Deterministic diagnostic logic
- Read-only operational analysis
- Firewall policy validation
- Route and NAT analysis
- Configuration consistency checks

## Workflow-Oriented Operations

- Guided operational workflows
- Validation-first configuration generation
- Change preview concepts
- Rollback planning concepts
- Vendor-aware operational logic

## Vendor Focus

Current primary focus:

- Fortinet FortiGate
- FortiAnalyzer

Active development:

- Palo Alto Networks PAN-OS integration

# Development Philosophy

The project prioritizes:

- operational safety
- deterministic validation
- vendor-aware workflows
- auditability
- engineering practicality
- incremental capability growth

The platform is intentionally being developed iteratively, with architectural flexibility and operational realism prioritized over rapid feature expansion.

---

# Public Repository Scope

This repository intentionally excludes proprietary implementation details, including:

- internal orchestration logic
- backend execution systems
- AI prompt systems
- production deployment architecture
- vendor command generation logic
- implementation-specific intellectual property

Public documentation focuses on architectural direction, workflow concepts, troubleshooting methodologies, and engineering approach.
